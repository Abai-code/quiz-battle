# №6 Практикалық жұмыс - Тестілеу есебі

**Жоба:** Quiz Battle
**Сынақ сервері:** Render.com
**Мәліметтер базасы:** Neon (PostgreSQL)

## 1. Орындалған жұмыстар
- [x] Жобаның құпия мәліметтері `.env` файлына көшірілді.
- [x] Парольдер Bcrypt арқылы шифрланды.
- [x] Мәліметтер базасы Neon (Cloud) платформасына көшірілді.
- [x] Backend Render.com-ға орналастырылды.
- [x] Frontend (React) Render.com Static Site ретінде іске қосылды.
- [x] Pagination (беттеу) жүйесі рейтинг бетіне енгізілді.

## 2. Тестілеу нәтижелері

| Функция | Тексеру қадамы | Нәтиже | Статус |
| :--- | :--- | :--- | :--- |
| Тіркелу | Жаңа қолданушы жасау | Сәтті, базада пайда болды | ✅ OK |
| Авторизация | Логин/Пароль арқылы кіру | JWT токен алынды | ✅ OK |
| Ойын | Сұрақтарды алу және ұпай жинау | Ұпайлар базада жаңартылды | ✅ OK |
| Рейтинг | Пагинацияны тексеру | "Next/Prev" батырмалары жұмыс істейді | ✅ OK |
| Аватар | Сурет жүктеу | Base64 форматында базаға сақталады | ✅ OK |

## 3. Қорытынды
Жоба сынақ серверінде тұрақты жұмыс істеп тұр. Барлық REST API endpoint-тері (200, 400, 401 қателерімен) тексерілді. Жоба деплойға толық дайын.
